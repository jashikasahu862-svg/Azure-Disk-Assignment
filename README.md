# Assignment: Azure Disk Storage Management

## Description
This project focuses on managing **Persistent Storage** with Azure Virtual Machines. It covers the end-to-end practical implementation of attaching, formatting, mounting, resizing, and migrating data disks across Azure VMs.

## Key Learning Outcomes
- **Disk Lifecycle:** Provisioning and attaching Azure managed disks to virtual machines.
- **Linux Integration:** Configuring disks using command-line utilities such as `lsblk`, `mkfs`, and `mount`.
- **Dynamic Resizing:** Extending disk capacity on-the-fly using `growpart` and `resize2fs` without downtime.
- **Portability:** Migrating persistent storage between different VMs while ensuring data integrity.

## Tasks Overview
- **Part A:** Theoretical coverage of Azure disk types, performance metrics (IOPS/Throughput), redundancy models, and security.
- **Part B:** Practical step-by-step implementation documentation with configuration screenshots.
